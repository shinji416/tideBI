# Fishing Tide BI サーフ強化版 + 地震津波アラート

## 追加機能

- 震度3以上の地震情報を監視
- 津波情報を確認
- 対象地域：静岡県、愛知県、三重県、和歌山県、福井県、石川県
- 地震・津波情報を検知した場合、画面全体に赤い警告を表示
- 警報音ON設定に対応
- iPhone/Safari対策として、警報音は設定画面の「警報音を有効化」を一度押す必要があります
- 設定画面に警報テストボタンを追加

## 注意

このアプリの地震・津波アラートは補助機能です。
釣り場では、気象庁、自治体、防災無線、緊急速報メール、海上保安庁などの公的情報を必ず優先してください。

## 使い方

1. GitHub Pagesへアップ
2. iPhoneのSafariで開く
3. ホーム画面に追加
4. 設定から「警報音を有効化」
5. 警報テストで音と表示を確認
