# Fishing Tide BI サーフ強化版

## 変更内容

- 地図画面を外し、釣り場候補リストから登録する方式に変更
- 県・種類・釣り場名を選ぶだけで緯度・経度を自動セット
- 愛知県・三重県・静岡県のサーフポイントを大幅追加
- 三重県は伊勢方面（有滝、村松、二見浦、宮川河口、五十鈴川河口、大淀、明和など）を追加
- ホーム画面の1ページ固定タイドグラフは維持

## 使い方

1. ホーム画面右上の「釣場」を押す
2. 県を選ぶ
3. 種類を選ぶ（サーフ、河口、港など）
4. 釣り場名を押す
5. 登録を押す

9地点が満杯の場合は、現在表示中の地点と入れ替えます。
